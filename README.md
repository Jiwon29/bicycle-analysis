# 🚴 따릉이 팝업광고 타겟팅 대시보드

서울시 공공자전거 따릉이의 이용 데이터를 분석하여 권종(정기권/일일권)별 맞춤형 팝업광고 타겟팅 전략을 도출하는 Streamlit 대시보드입니다.

## 주요 분석 내용
- 정기권/일일권 이용 지역 TOP 100 — 서울 지도 시각화
- 권종별 성별 구성 파이차트
- 권종별 이용건수 및 월별 추이
- 연령대별 이용 패턴
- 데이터 기반 팝업광고 멘트 타겟팅 전략 12종

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## DB 구조 (자전거분석_slim.db)
원본 122MB DB에서 대시보드에 필요한 집계 결과만 추출하여 **1MB**로 경량화했습니다.

| 테이블 | 설명 |
|--------|------|
| 지역집계 | 대여소별 권종 이용건수 (위도/경도 포함) |
| 성별집계 | 권종 × 성별 이용건수 |
| 이용건수집계 | 권종별 총 이용건수 |
| 연령대집계 | 권종 × 연령대 이용건수 |
| 월별집계 | 월별 권종 이용 추이 |
| 팝업광고타겟팅 | 타겟별 팝업 멘트 및 인사이트 |
| 대여소 | 대여소 기본 정보 (위도/경도) |
